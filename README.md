# MovieApp
Movie App is an application where you can follow movie trends, search for the movie you like and add the ones you like to your favorites.

# Libraries Used

+ MVVM + Clean Architecture
+ Retrofit
+ Hilt
+ Room
+ Navigation Component
+ ViewModel
+ Coroutines
+ Flow
